        <h2 style="text-align: center; color: #2E7D32; margin-bottom: 20px;">LESSON 2</h2>
        <p>
            Stylesheets are where CSS rules are written and stored. They tell the browser how to display HTML elements. CSS can be applied in three main ways: inline, internal, and external. Inline styles are written directly inside an HTML tag using the style attribute. Internal styles are placed within a &lt;style&gt; tag inside the &lt;head&gt; section of an HTML document, and external styles are written in a separate .css file and linked to the HTML file using the &lt;link&gt; tag. External stylesheets are the most efficient and professional way to manage large websites because they separate style from structure and make it easy to update the design of multiple pages at once.
        </p>

        <h2 style="text-align: center; color: #2E7D32; margin-bottom: 20px;">LESSON 3</h2>
        <p>
            In CSS, the display property controls how an element is positioned and behaves in a layout. Block-level elements (like &lt;div&gt;, &lt;p&gt;, and &lt;h1&gt;) begin on a new line and stretch across the entire available width, making them useful for dividing and structuring sections of a page. Inline elements (such as &lt;span&gt;, &lt;a&gt;, and &lt;strong&gt;) stay on the same line and only occupy as much space as their content requires. Knowing the difference between these display types is key to creating well-structured and properly aligned webpages. CSS also lets developers modify an element’s display mode, offering greater flexibility and control over layout design.
